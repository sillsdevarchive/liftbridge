Overview

Lift Bridge is a plugin for FieldWorks Language Explorer which permits sharing of LIFT dictionary data with remote colleagues such as WeSay.

Usage

See documentation for details on how to use Lift Bridge.

Downloads

Lift Bridge is now available for use with only one version of FieldWorks, which is the 7 series (7.1 or higher). The older version that worked with FieldWorks 6.0.6 is no longer available, since the older version of FieldWorks is not getting required bug fixes in it that are needed to run Lift Bridge properly.
